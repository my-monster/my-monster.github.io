# X射线产生机制

高速电子流和靶面碰撞后会产生两种X射线，一种是轫致辐射（连续X射线），一种是特征辐射。同时也会有碰撞损失，转换为能量。

# 连续X射线

连续X射线是高速电子流和原子核相互作用，产生韧致辐射。

# 特征X射线

特征X射线能量和管电压无关，只和电子轨道之间的能级差有关系。
