    torch.nn.Flatten()的参数实例

    # 2、torch (opens new window).nn.Flatten()的参数实例

    # 默认参数
import torch
a = torch.randn(8,3,64,64)
F = torch.nn.Flatten()
a1 = F(a)

a的大小：
torch.Size([8, 3, 64, 64])

a1的大小：
torch.Size([8, 12288])
默认将第0维保留下来，其余拍成一维
